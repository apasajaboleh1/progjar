untuk file chatserver.py dan chatclient.py
jadi cara kerja program ini adalah
1. untuk username atau nama jangan ada spasinya

step by stepnya 
1. kita mengetikkan kata loginnya
2. kita menggetikan username / nama yang ingin kita pilih
3. kita bs langsung menggunakan command seperti:
	A. list untuk melihat siapa saja yang aktif list
	B. sendall untuk membroadcast ke semuanya yang aktif pada saat itu formatnya sendall [spasi] [pesan]
	C. sendto untuk membuat private message dengan orang yang kita tuju formatnya sendto [spasi] [nama orangnya] [spasi] [pesan] 
4. kalau mau langgsung keluar tinggal Ctrl + C
